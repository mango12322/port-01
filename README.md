## 1. 프로젝트 개요

본 프로젝트는 여행 콘텐츠를 주제로 한 **반응형 웹사이트 퍼블리싱 프로젝트**입니다.

사용자가 다양한 여행 정보를 직관적으로 탐색할 수 있도록 UI를 구성하고,

모던 레이아웃과 인터랙션을 구현하는 것을 목표로 제작했습니다.

---

## 2. 사용 기술

- HTML5 (Semantic Markup)
- CSS3
- Flexbox / Layout 구성
- JavaScript
- jQuery
- Slick Slider
- 반응형 웹 (Media Query)

---

## 3. 주요 기능

### ① 반응형 웹 구현

PC, 모바일 환경에서 레이아웃이 자연스럽게 변경되도록

Media Query를 활용해 반응형 웹을 구현했습니다.

---

### ② 슬라이드 기능 구현

메인 배너 영역에 슬라이더를 적용하여 여러 콘텐츠를 시각적으로 강조했습니다.

사용 라이브러리

- Slick Carousel

자동 슬라이드 및 여러 콘텐츠를 한 번에 보여주는 기능을 구현했습니다.

---

### ③ 상단 네비게이션 메뉴

상단 메뉴를 통해 페이지 내 각 섹션으로 이동하도록 구성했습니다.

또한 사용자 편의를 위해 스크롤 이동 애니메이션을 구현했습니다.

---

### ④ 스크롤 인터랙션

스크롤 위치에 따라 콘텐츠가 자연스럽게 나타나는 인터랙션을 구현하여

사용자 경험을 개선했습니다.

---

## 4. 프로젝트 진행 과정

1. 와이어프레임 설계
2. HTML 시멘틱 구조 설계
3. CSS 레이아웃 구현 (Flex 기반)
4. 슬라이더 및 인터랙션 기능 구현
5. 반응형 레이아웃 적용 (모바일)

---

## 5. 문제 해결 경험

슬라이더 구현 과정에서 여러 개의 콘텐츠가 동시에 표시되도록 설정하는 과정에서

레이아웃이 깨지는 문제가 발생했습니다.

이를 해결하기 위해 슬라이더 옵션과 CSS flex 레이아웃을 함께 조정하여

콘텐츠 간 간격과 정렬 문제를 해결했습니다.

---
